<Note>
  To mint subnames from the Namespace platform, ENS Widget, or using our SDK,
  you must first activate your ENS Name using the Wizard.
</Note>

## How to Activate an ENS Name

<Steps>
  <Step title="Open the app and start activation">
    1. Visit <a href="https://app.namespace.ninja/">app.namespace.ninja</a>.
    2. Connect the wallet that owns the ENS name you want to activate.
    3. Click the "Account" icon in the navigation bar.
    4. Open the "Wizard" tab and click "List".
    <img src="/assets/activate-ens-name-1.png" alt="Open Wizard and start activation" />
  </Step>

  <Step title="Select your ENS name">
    Choose the ENS name to activate (for example, <code>ninjademo.eth</code>).
    <img src="/assets/activate-ens-name-2.png" alt="Select the ENS name to activate" />
  </Step>

  <Step title="Choose activation option">
    - <strong>Ethereum</strong>: Issue subnames on Ethereum.
    - <strong>L2s (Base, Optimism, etc.)</strong>: Issue subnames on supported L2s.

    This guide uses Base as the example.
    <img src="/assets/activate-ens-name-3.png" alt="Choose L2 for subnames (Base example)" />
  </Step>

  <Step title="Configure L2 collection">
    Select "Issue L2 Subnames", then:
    - Name your collection and set a symbol.
    - Decide on burnability and rental options.
    <img src="/assets/activate-ens-name-4.png" alt="Configure collection name, symbol, and options" />
  </Step>

  <Step title="Set mint pricing">
    Define how subnames are priced:
    - Base price
    - Length-based pricing
    - Special rules (numbers, letters, emoji)
    <img src="/assets/activate-ens-name-5.png" alt="Configure mint pricing options" />
  </Step>

  <Step title="Reserve or blacklist subnames (optional)">
    Protect key names or block unwanted ones.
    <img src="/assets/activate-ens-name-6.png" alt="Reserve or blacklist subnames" />
  </Step>

  <Step title="Set a minting deadline (optional)">
    Make the collection time-bound if desired.
    <img src="/assets/activate-ens-name-7.png" alt="Set an optional minting deadline" />
  </Step>

  <Step title="Configure allowlist options">
    Restrict minting to specific wallets or allow free mints for approved addresses.
    <img src="/assets/activate-ens-name-8.png" alt="Allowlist and free mint options" />
  </Step>

  <Step title="Enable token-gated minting (optional)">
    Require holders of an ERC-20 or ERC-721 token to mint.
    <img src="/assets/activate-ens-name-9.png" alt="Enable token-gated minting" />
  </Step>

  <Step title="List your name">
    When setup is complete, click "List Name".
  </Step>

  <Step title="Set the Namespace Hybrid Resolver">
    Set the resolver to Namespace Hybrid Resolver.
    <img src="/assets/activate-ens-name-10.png" alt="Set resolver to Namespace Hybrid Resolver" />
  </Step>

  <Step title="Configure resolver settings">
    Follow the prompts to complete resolver configuration.
    <img src="/assets/activate-ens-name-11.png" alt="Configure resolver settings" />
  </Step>

  <Step title="All set!">
    Congratulations â€” your ENS name is now activated.
    <img src="/assets/activate-ens-name-12.png" alt="Activation complete" />
  </Step>
</Steps>

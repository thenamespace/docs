---
title: "Delete a Subname"
description: "Permanently remove a subname from the system. Requires API key authentication."
openapi: "/openapi/offchain-openapi.json DELETE /api/v1/subnames/{fullSubname}"
---

<Warning>
  This endpoint permanently deletes a subname. API key authentication is
  required via the `x-auth-token` header.
</Warning>

## Path Parameters

<ParamField path="fullSubname" type="string" required>
  The complete subname to delete (e.g., `alice.oppunk.eth`)
</ParamField>

## Response Codes

- **201**: Subname deleted successfully
- **401**: API key not present or invalid
- **404**: Subname not found

<Note>
  Ensure you have the proper permissions to delete the subname. Only authorized
  API keys can perform deletion operations.
</Note>

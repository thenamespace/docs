---
title: "Create a New Subname"
description: "Register or update a subname with text records, addresses, and metadata. Requires API key authentication."
openapi: "/openapi/offchain-openapi.json POST /api/v1/subnames"
---

<Info>
  This endpoint creates a new subname or updates an existing subname. API key
  authentication is required via the `x-auth-token` header.
</Info>

## Request Examples

<CodeGroup>

```javascript Basic Subname
{
  "parentName": "oppunk.eth",
  "label": "alice",
  "owner": "0x1234567890123456789012345678901234567890"
}
```

```javascript With Text Records
{
  "parentName": "oppunk.eth",
  "label": "alice",
  "owner": "0x1234567890123456789012345678901234567890",
  "texts": [
    {
      "key": "description",
      "value": "Alice's ENS subname"
    },
    {
      "key": "url",
      "value": "https://alice.example.com"
    }
  ]
}
```

```javascript With Address Records
{
  "parentName": "oppunk.eth",
  "label": "alice",
  "owner": "0x1234567890123456789012345678901234567890",
  "addresses": [
    {
      "coin": 60,
      "value": "0x1234567890123456789012345678901234567890"
    },
    {
      "coin": 0,
      "value": "1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa"
    }
  ]
}
```

</CodeGroup>

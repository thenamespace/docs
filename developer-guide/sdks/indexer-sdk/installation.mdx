---
description: >-
  indexer-manager is a lightweight wrapper library around the Namespace Indexer,
  designed to simplify searching and filtering of minted subnames across
  multiple chains
---

You can install the library using either **npm** or **yarn**:

<Tabs>
<Tab title="Npm">
```bash
npm install @namespacesdk/indexer-manager
```
</Tab>
<Tab title="Yarn">

```bash
yarn add @namespacesdk/indexer-manager
```

</Tab>
</Tabs>

### Usage

To create an instance of the `IndexerClient`, import the factory method from the package:

```typescript
import {
  createIndexerClient,
  IndexerClient,
} from "@namespacesdk/indexer-manager";

const indexer = createIndexerClient();
```

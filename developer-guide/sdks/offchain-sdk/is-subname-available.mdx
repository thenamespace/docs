---
title: "Check Subname Availability"
description: "Determine if a specific ENS subname is available for registration."
sidebarTitle: "Subname Availability"
---

Use `isSubnameAvailable` to check if a subname can be registered.

```typescript
const response = client.isSubnameAvailable("ns.myensname.eth");
const isAvailable = response.isAvailable;
```

## Return Type

```typescript
export interface GetAvailableResponse {
  isAvailable: boolean;
}
```

- `isAvailable`: `true` if the subname is available, `false` if already taken.

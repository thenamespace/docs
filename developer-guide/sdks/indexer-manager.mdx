## Indexer Client

IndexerClient is a library for interacting with indexed data from Namespace smart contracts. It is built on top of Axios and provides a convenient API for gathering subnames data from Namespace.

The `createIndexerClient` function sets up an Indexer Client.

```typescript
import { createIndexerClient } from "@namespacesdk/indexer-manager";

const client = createIndexerClient({});
```
